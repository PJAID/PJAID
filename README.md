Aplikacja mobilna do zgłaszania i zarządzania awariami urządzeń w ramach systemu PJAID.

MVP (Minimum Viable Product)
- Rejestracja zgłoszenia awarii
- Skanowanie kodu QR
- Lista zgłoszeń użytkownika
- Szczegóły zgłoszenia
